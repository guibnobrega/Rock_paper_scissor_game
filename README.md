# Rock Paper Scissors Game

Simple Rock, Paper, Scissors game implemented in Python.

**Description**
- Educational project that implements the classic Rock, Paper, Scissors rules.
- Main files: [data.py](data.py), [functions.py](functions.py), [main.py](main.py).

**Prerequisites**
- Python 3.8 or newer.

**Installation**
1. Clone the repository:

```bash
git clone <REPO-URL>
cd Rock_paper_scissor_game
```

2. (Optional) Create and activate a virtual environment:

```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
```

**Running the game**

Run the main script:

```bash
python main.py
```

Follow the on-screen prompts to play.

**Project structure**
- [data.py](data.py): game data and constants.
- [functions.py](functions.py): game logic (move comparison, validation).
- [main.py](main.py): command-line interface and main loop.
